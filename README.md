# 💰 FinTrack - Finance Dashboard UI

## 📌 Overview

FinTrack is a modern and interactive Finance Dashboard built using React and Tailwind CSS.  
It helps users track financial activities, visualize spending patterns, and gain useful insights through a clean and user-friendly interface.

This project focuses on frontend architecture, UI/UX design, and efficient state management.

---

## ✨ Features

### 📊 Dashboard Overview
- Total Balance, Income, and Expense summary cards
- Responsive and modern UI
- Clean layout with proper spacing and design hierarchy

### 📈 Data Visualization
- Line Chart for financial trends
- Pie Chart for spending breakdown by category
- Real-time updates on data changes

### 📋 Transactions Management
- List of all transactions with:
  - Date
  - Category
  - Amount
  - Type (Income/Expense)
- Search functionality (by category)
- Filter (Income / Expense / All)

### 🔐 Role-Based UI (Frontend Simulation)
- Viewer → Can only view data
- Admin → Can add transactions

### ➕ Add Transaction (Admin Only)
- Add new income or expense
- Instantly updates dashboard, charts, and insights

### 💡 Insights Section
- Top spending category
- Total number of transactions
- Dynamic insights based on data

### 🌙 Dark Mode
- Toggle between light and dark themes
- Theme preference saved using localStorage

## 🛠️ Tech Stack

- ⚛️ React.js (Vite)
- 🎨 Tailwind CSS
- 📊 Recharts (Charts & Visualization)
- 🧠 JavaScript (ES6+)
